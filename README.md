# FullSysAdmin

To be able to run smoothly this SystemInfo App
Following requisits:

*** Currently only available for Arch Linux (and derivatives ??? ... but not tested out of plain Arch)
Following apps need to be installed
- pacman-contrib package needed for checkupdates
- yay : this to use the yay statistics
- expac : Arch utility to get latest installed packages
- ranger: a cli-based file manager
- dysk : better tool for installed disk info (in stead off <df>)
- lynis: a complete system diagnostic tool (checks on troyans, viruses, ...)

The Freemem bash script has be be downloaded into ~/bin
